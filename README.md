# Angular2-Starter
Starter for Angular 2 based on scotch.io lesson on Angular2  by Chris Sevilleja

Uses System.js to load app

Instructions :
 <code>npm install</code>
 
 Serve it using :
 
 <code>npm start</code>
 
 Compile Typescript using :
 <code>npm run tsc</code>
 
 or to watch 
 
  <code>npm run tsc :w</code>
